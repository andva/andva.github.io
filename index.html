Imperfect shadow mapping

The project implemented a report describing a technique for simulating low frequency soft shadows in a scene by using shadow maps and a point representation of the scene. The implementation was based on the paper by Ritschel et. al. This allowed for increasing the light source count in the scene and make the lightning converge towards a real representation of area lights while still running in real time. To add more high frequency shadows we used a screen space ambient occlusion algortihm.

I implemented most of the imperfect shadow mapping algorithm while the other project member worked on the screen space ambient occlusion. The code was written in C++ using modern OpenGL to generate points on triangles, create shadow mapps and merge the results together.

Physically Based Simulation of Fire

In this projected I created physically based simulation of Fire with 4 other students. The implementation is largely based on the paper Physically Based Modeling and Animation of Fire by Nguyen et al.

We simulate Fire by using two sets of incompressible flow equations representing the unignited fuel (blue flame) and the hot gaseous products (yellow-orange flame). These equations are coupled together using a ghost fluid method, the interface separating the fluids are tracked using level set methods. We also track the temperature across the simulation to simulate external forces such as buoyancy, and to be able to render the simulation with a visually accurate result.

Depth Of Field by Simulated Heat Diffusion

In this project I created together with 2 other students a post-processing effect on the GPU, which simulates the depth of field using the Diffusion equation. The project is based on the Pixar report Interactive Depth of Field Using Simulated Diffusion on a GPU.

There is a few different methods to post-process depth of field, most of them suffers from color bleeding, e.g. when the color from a out-of-focus object which lie behind a in-focus object, can be seen on the focused object. This method solves this problem by solving the Diffusion equitation using a ADI method. This results in a tri-diagonal linear system which can efficiently be solved on the GPU using various methods, we used the Jacobi method. The whole process was computed using the GLSL fragment shader.

Even though the method solves the color bleeding problem, it also add other artefacts.

Interactive real time Cloth Simulation

This is a mass-spring cloth simulation, where the user can grab the cloth, tear it, cut it, etc. This project was made in the course TNM085 Modelling and simulation with 4 other students

It is quite a simple simulation model, where the cloth is made of lots of particles, which are aligned in a square-structured grid to each other. The grid is then held together by spring connections between the particles. The simulation uses the leapfrog integration method to calculate a new position over time.

I worked mostly with the tearing functionality, the integration scheme and a simple surface-collision algorithm.
