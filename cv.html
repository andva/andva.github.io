Experience

2014.03 – Master's thesis – EA Frostbite, Stockholm
I did my master thesis at Frostbite in Stockholm. The project was an effort to create a interactive light and fast scene representation for light and occlusion calculations on the GPU. During the project, I created a system for converting geometry into a voxel grid in real time using Direct X and HLSL. I also made a implementation of ambient occlusion, a lightning computation for coarse approximations of non-directional lightning in a scene to showcase the benefits of my data structure.

2012.08 – 2014.01 Programming mentor - Linköping University
I was involved in the founding of an initiative to help students with everything from the basics of programming to memory handling in C++, code structure and patterns for lean and extendable code. The initiative was highly appreciated by both students and teachers and our group are now supported by the department of science and technology at Linköping University.

2012.06 – 2012.08 Software developer – Moving Picture Company
I worked on the software team for the movie Guardians of The Galaxy where I incorporated the Pixar OpenSubdiv library and made sure that it was supported in all parts of the pipeline. I also worked on improving the rigid body collision system by creating a prototype for a newsystem for collisions between concave objects using a volume representation of the objects and improved their Lua based rendering scripts.

2013.5 – 2013.6 Software developer – SAAB Dynamics
I wrote a plugin for Eclipse that retrieved an image from memory during GDB sessions. The tool made it possible for the developers to debug images during development, without doing any modifications in the code. The tool was created in Java and had an interface for entering the memory address to the image, and a window for doing simple image operations on the image like zooming, panning and writing pixel values when zoomed in enough.

2013.1 – 2013.5 Software developer – C-Research
I worked for a research team connected to the university, writing code for a node based scientific visualization tool. I worked in C++ writing nodes while implementing and linking external libraries using CMake.
